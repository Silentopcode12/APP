# Docker Mastery Checklists

## Core Skills
- Build images from Dockerfiles
- Understand layers and caching
- Run, stop, and remove containers
- Inspect containers and images
- Persist data with volumes
- Use bind mounts for local dev
- Create and use networks
- Publish ports correctly
- Use Docker Compose for multi-service apps

## Debugging And Ops
- Read container logs
- Use `docker exec` for live debugging
- Use `docker inspect` to validate config
- Diagnose networking issues
- Diagnose volume mount issues
- Understand exit codes and healthchecks

## Performance
- Reduce image size
- Reduce build time
- Control CPU and memory
- Use `.dockerignore` effectively

## Security
- Run as non-root
- Avoid secrets in images
- Use minimal base images
- Scan images for vulnerabilities

## Distribution
- Tag images correctly
- Push and pull from registries
- Use image digests
